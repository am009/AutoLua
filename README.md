# AutoLua

Android自动化，集成Lua解释器，支持坐标点击，查找元素，界面滚动。

特点：

1. 集成WebView，Web技术构建代码编辑器
2. 集成Lua引擎，使用原生Lua和JNI，不引入其他Lua-Java接口。


功能：

1. 定时执行
2. 多样化控制
3. 可以查看错误信息，方便定位错误。
4. 中英文界面支持
